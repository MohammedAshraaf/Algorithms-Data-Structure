what is C# language

C# It's an object oriented programming language that help programmers to build variety of secure and strong applications

you can use it to create windows applications , data base application and more

C#'s syntax is simple and easy to learn if u know "JAVA, C++, or C" you can begin to work productively in C# within a very short time.

C# supports the concepts of encapsulation, inheritance, and polymorphism.

C# is part of the .NET framework,so it is compiled to Microsoft Intermediate Language (MSIL), which is a language similar to Java's bytecode.
