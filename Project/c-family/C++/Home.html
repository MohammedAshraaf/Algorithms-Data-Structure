What is C++ language

C++ is one of the four languages used to develop apps for Windows 8,

A large part of MySQL and Adobe were written in C++.
In 2000, Microsoft released a version of C++ compatible with dot-net.

C++ has a lot of low level machine commands to let you control video cards, audio cards So, it is popular for gaming development. It used to be the only option for multiplayer games over networks and 3D games because it offered greater control over audio and video output.

C++ can be compiled to run directly on the processor.

C++ is the basis of languages like Java, so understanding its logic helps you understand the concept of many other languages.
