what is PHP Language

PHP iS a programming language to develop dynamic websites

It offers web developers a large selection of instruments
it has become the basis for many web applications, it's server side scripting language, it means that all the processing happens before any code get sent to the browser

so u don't have to worry about someone see your php code, because it processed in your server all they're going to see is the output.
